```swift
// Filtering logic to reuse:
private var filteredItems: [ClipboardItem] {
    let sorted = items.sorted { a, b in
        if a.isPinned != b.isPinned { return a.isPinned }
        return a.timestamp > b.timestamp
    }
    if searchText.isEmpty { return sorted }
    return sorted.filter { $0.content.localizedCaseInsensitiveContains(searchText) }
}
```

  <action>
Create three new SwiftUI view files:

**1. VisualEffectView.swift** — NSViewRepresentable for frosted glass:
- Struct `VisualEffectView: NSViewRepresentable` with configurable `material` (default: `.hudWindow`) and `blendingMode` (default: `.behindWindow`)
- In `makeNSView`: create NSVisualEffectView, set material, blendingMode, and CRITICALLY set `state = .active` (required for `.accessory` activation policy apps — without this, blur renders as flat gray)
- `updateNSView`: update material and blendingMode

**2. OverlayItemRow.swift** — Individual row in overlay list:
- Struct `OverlayItemRow: View` with `let item: ClipboardItem` property
- Display: truncated content preview (first 100 chars, single line with `.lineLimit(1)`), source app name in secondary caption if available, relative timestamp
- Use `DisplayFormatter.format(item.content, maxLength: 100, patterns: [])` for preview text (no redaction patterns needed in overlay — keep it simple)
- Style: `.padding(.horizontal, 12).padding(.vertical, 6)`, clear background (list selection handles highlighting)

**3. ClipboardOverlayView.swift** — Root overlay SwiftUI view:
- Struct `ClipboardOverlayView: View` with `@Environment(\.modelContext)`, `@Query(sort: \ClipboardItem.timestamp, order: .reverse) private var items: [ClipboardItem]`
- `@State private var searchText = ""`
- `@State private var selectedID: ClipboardItem.ID?`
- `@FocusState private var searchFocused: Bool`
- `@State private var showContent = false` (for animation)
- `filteredItems` computed property: same logic as ClipboardPopup (pinned first, then by timestamp desc, filtered by searchText)
- Auto-select first item: use `.onChange(of: filteredItems)` and `.onAppear` to set `selectedID = filteredItems.first?.id` when list changes

Layout (top to bottom in a VStack(spacing: 0)):
  1. Search TextField "Search clipboard..." with `.focused($searchFocused)`, padding 12
  2. Divider
  3. List with `selection: $selectedID` binding, showing `OverlayItemRow` for each filtered item, tagged with `.tag(item.id)`
  4. Bottom bar: item count label (e.g. "42 items"), right-aligned

Keyboard handling:
  - `.onKeyPress(.return)`: if selectedID exists and matches a filtered item, call `OverlayWindowController.shared.pasteAndHide(content: item.content)`, return `.handled`
  - `.onKeyPress(.escape)`: call `OverlayWindowController.shared.hide()`, return `.handled`

Focus: `.onAppear { searchFocused = true; showContent = true }`

Animation wrapper: Wrap the entire VStack in a ZStack with `VisualEffectView()` as background. Apply `.clipShape(RoundedRectangle(cornerRadius: 12))` to the ZStack. Wrap the content VStack in `if showContent` with `.transition(.opacity.combined(with: .scale(scale: 0.95)))` and `.animation(.easeOut(duration: 0.15), value: showContent)`.

Frame: `.frame(width: 640, height: 400)`

Reset state when overlay opens: Add a notification observer for a custom `Notification.Name("overlayWillShow")` that resets `searchText = ""`, `selectedID = filteredItems.first?.id`, `searchFocused = true`, `showContent = true`. The OverlayWindowController.show() will post this notification.
