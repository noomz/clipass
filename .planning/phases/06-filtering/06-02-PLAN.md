- **Real-time regex validation** (copy from RuleEditorView):
  ```swift
  @State private var patternError: String?
  
  private var isPatternValid: Bool {
      guard !pattern.isEmpty else { return true }
      do {
          _ = try Regex(pattern)
          return true
      } catch {
          return false
      }
  }
  
  private func validatePattern(_ value: String) {
      guard !value.isEmpty else {
          patternError = nil
          return
      }
      do {
          _ = try Regex(value)
          patternError = nil
      } catch {
          patternError = "Invalid regex: \(error.localizedDescription)"
      }
  }
  ```
- Show patternError in red caption below pattern field
- isEnabled toggle
- Cancel/Save buttons in header
- Save disabled if name empty, pattern empty, or !isPatternValid
- **Test section** like RuleEditorView: testInput field, show if content would be ignored (matches pattern)
- On save: update existing or insert new IgnoredPattern
- **Important:** After save, call ClipboardMonitor.invalidatePatternCache() if editing existing pattern
