Modify ClipboardMonitor.poll() to check ignored apps and patterns before storing clipboard content.

**Changes:**

1. Add a cached regex store to avoid recompiling on every poll:
```swift
private var cachedPatternRegexes: [UUID: Regex<Substring>] = [:]
```

2. Add helper method to get or compile regex:
```swift
private func getRegex(for pattern: IgnoredPattern) -> Regex<Substring>? {
    if let cached = cachedPatternRegexes[pattern.id] {
        return cached
    }
    guard let regex = try? Regex(pattern.pattern) else { return nil }
    cachedPatternRegexes[pattern.id] = regex
    return regex
}
```

3. In poll(), after getting sourceApp and before the main async block, add checks:

**Check ignored apps** (before storing):
```swift
// Check ignored apps
let appDescriptor = FetchDescriptor<IgnoredApp>()
if let ignoredApps = try? context.fetch(appDescriptor),
   let sourceApp = sourceApp {
    for ignoredApp in ignoredApps where ignoredApp.isEnabled {
        if sourceApp == ignoredApp.bundleId || sourceApp.contains(ignoredApp.bundleId) {
            return // Skip ignored app
        }
    }
}
```

**Check ignored patterns** (before storing):
```swift
// Check ignored patterns
let patternDescriptor = FetchDescriptor<IgnoredPattern>()
if let ignoredPatterns = try? context.fetch(patternDescriptor) {
    for ignoredPattern in ignoredPatterns where ignoredPattern.isEnabled {
        if let regex = getRegex(for: ignoredPattern),
           transformedContent.contains(regex) {
            return // Skip content matching ignore pattern
        }
    }
}
```

Place these checks AFTER transforms are applied but BEFORE the duplicate check and storage. The order is:
1. Check ignored types (existing)
2. Get content
3. Get sourceApp
4. Apply transforms (existing)
5. **NEW: Check ignored apps** — if match, return early
6. **NEW: Check ignored patterns** — if match, return early
7. Skip duplicate check (existing)
8. Store item (existing)

Also add a method to invalidate cached regexes when patterns change (will be called from UI later):
```swift
func invalidatePatternCache() {
    cachedPatternRegexes.removeAll()
}
```
