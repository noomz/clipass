  <action>
Refactor the `SettingsView` `TabView` to use the macOS 15+ `Tab` API with fallback to the current `.tabItem` approach for macOS 14.

**For macOS 15+ (primary path):** Use the new `Tab` initializer which renders as proper toolbar tabs with large SF Symbol icons — the native macOS Settings/Preferences look:

```swift
struct SettingsView: View {
    var body: some View {
        Group {
            if #available(macOS 15.0, *) {
                tabViewModern
            } else {
                tabViewLegacy
            }
        }
        .frame(width: 550, height: 450)
        .onDisappear {
            NSApp.setActivationPolicy(.accessory)
        }
    }
}
```

**macOS 15+ view** — Use `Tab` with `systemImage` parameter. Use `.tabViewStyle(.sidebarAdaptable)` which provides the modern macOS Settings sidebar that adapts to toolbar tabs:

```swift
@available(macOS 15.0, *)
private var tabViewModern: some View {
    TabView {
        Tab("General", systemImage: "gear") {
            GeneralSettingsView()
        }
        Tab("Transforms", systemImage: "wand.and.stars") {
            RulesView()
        }
        Tab("Automation", systemImage: "bolt") {
            HooksView()
        }
        Tab("Filtering", systemImage: "line.3.horizontal.decrease.circle") {
            FilteringSettingsView()
        }
        Tab("Display", systemImage: "text.alignleft") {
            DisplaySettingsView()
        }
    }
    .tabViewStyle(.sidebarAdaptable)
}
```

**macOS 14 fallback** — Keep the existing `.tabItem` approach (which renders as toolbar tabs on macOS 14):

```swift
private var tabViewLegacy: some View {
    TabView {
        GeneralSettingsView()
            .tabItem { Label("General", systemImage: "gear") }
        RulesView()
            .tabItem { Label("Transforms", systemImage: "wand.and.stars") }
        HooksView()
            .tabItem { Label("Automation", systemImage: "bolt") }

            .tabItem { Label("Filtering", systemImage: "line.3.horizontal.decrease.circle") }
        DisplaySettingsView()
            .tabItem { Label("Display", systemImage: "text.alignleft") }
    }
}
```

**Window size change:** Increase from `(width: 500, height: 400)` to `(width: 550, height: 450)` to give tabs more room and prevent content crowding.

Keep the existing `GeneralSettingsView`, `FilteringSettingsView` structs, and all other code in the file unchanged.
