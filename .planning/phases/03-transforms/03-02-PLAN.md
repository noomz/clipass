1. Add default rules creation to TransformEngine:
   - Method: `createDefaultRulesIfNeeded(context: ModelContext)`
   - Check if any rules exist, if not create defaults
   - Call this on app launch (in ClipboardPopupContainer or similar)

2. Default rules to create:
   a. "Strip trailing whitespace"
      - pattern: "\\s+$"
      - replacement: ""
      - sourceAppFilter: nil (all apps)
      - order: 100

   b. "Strip Terminal trailing whitespace"
      - pattern: "\\s+$"
      - replacement: ""
      - sourceAppFilter: "com.apple.Terminal"
      - order: 10

   c. "Normalize line endings"
      - pattern: "\\r\\n"
      - replacement: "\\n"
      - sourceAppFilter: nil
      - order: 200

3. Add "Reset to Defaults" functionality:
   - Method: `resetToDefaultRules(context: ModelContext)`
   - Delete all existing rules
   - Call createDefaultRulesIfNeeded

**Avoid:**
- Overwriting user rules on every launch — only create if no rules exist
- Too many default rules — start with 2-3 essential ones
