<task type="auto">
  <name>Task 2: Create HookEngine service</name>
  <files>clipass/Services/HookEngine.swift</files>
  <action>Create HookEngine class with:
  - private var modelContext: ModelContext?
  - func setModelContext(_ context: ModelContext)
  - func executeHooks(content: String, sourceApp: String?)

  executeHooks should:
  1. Fetch all enabled hooks sorted by order
  2. For each hook: check sourceAppFilter (nil = all apps), check pattern match (empty = match all)
  3. If matches, run command asynchronously using Process:
     - Use /bin/zsh -c "command" pattern
     - Set environment variable CLIPASS_CONTENT with the clipboard content
     - Set environment variable CLIPASS_SOURCE_APP with source app (or empty)
     - Run on background queue (don't block UI)
     - Log errors to console but don't crash

  Use Process (not deprecated NSTask). Run fire-and-forget - don't wait for completion.

  Pattern matching: use Swift Regex, catch errors for invalid patterns.</action>
  <verify>Build succeeds, no compiler errors</verify>
  <done>HookEngine.swift with executeHooks that runs matched hooks via Process</done>
</task>
